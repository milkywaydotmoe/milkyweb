These are the browsers that I have tested my site on. Testing criteria stolen from April

  • 5.0: Minimum score indicating that the site is usable
  • 7.0: Renders the website as expected, with very minor problems.
  • 7.5: Minimum score indicating the website renders as expected.
  • 9.0: Supports this website almost perfectly, with very minor problems
  • 10.0: Supports this website perfectly.

Testing Priority Rating Scale:

  • Very Low: Browser is the lowest priority for testing. May go an indefinite amount of time without another test.
  • Low: Browser is low priority for testing. Usually tested only every couple of months.
  • Medium: Browser is medium priority for testing. Usually tested every couple of updates and updates that make large changes.
  • High: Browser is high priority for testing. Usually tested on most moderate-sized updates.
  • Very High: Browser is very high priority for testing. Usually tested on moderate-sized updates and some smaller updates.
  • Top: Browser is one of my primary browsers and is the highest priority for testing. Usually tested every update.
